Import Annotations

Annotated books are displayed with a background color representing the quality of metadata matched with books in your library.

Matched metadata Annotations will be added…
Title, Author, UUID to matched library book automatically
Title, Author, Filename to matched library book automatically
Title, Author to matched library book after confirmation
No metadata matches to currently selected library book after confirmation

 

To Preview a book's annotations, select the book, then click Preview.

To import annotations from all checked books, click Import Annotations.
